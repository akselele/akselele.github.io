LoLTV

A coverage site for League of Legends esports - Svelte.js and Node.js

Being passionated by esports, I always thought that League of Legends esports was poorly covered, with only sites like Liquipedia and lol.fandom having detailled data available, while lolesports feels clunky.

In comparison to CS:GO, where HLTV.org has complete match data, even from back in 2012.

This project is still in early stages, with only a small portion of the back-end already working.

There is a Python script which gathers all information from games (2013-present) and inserts them into the database; the 'match' table in the database is already complete.

BetterMessengerTT

A Chrome (and Firefox?) addon for Twitch emotes on Facebook Messenger.

This project was my first project made during quarantine, where I was very bored. I thought to myself "hey, wouldn't it be fun to use Twitch emotes in Messenger with my friends like we use emojis?"

So I got to work.

Results:

  • Full support for emotes from frankerfacez in Facebook Messenger (sadly, only on the standalone webapp).
  • First dive into Javascript development

This project is no longer updated and probably no longer works.
